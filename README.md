# HABITUS_Furniture.Website
 
HABITUS is a front-end furniture website built with HTML, CSS, and JavaScript. The goal is to give the user a simple and clear shopping experience. The layout uses percentage based units to support responsive behavior. The design includes functional pages and a dark mode option.

Features
Responsive layout using percentage based CSS,
 Product details page,
 Dark mode,
 Clean folder structure,
 Simple navigation,
 Static front-end build with no backend.

How to Run:
Download the project folder,
 Open the main HTML file in your browser,
 Browse the pages,
 Test the layout on different screen sizes.

Project Structure:
index.html,
 other html files,
 assets folder for images,
 css folder for styles,
 js folder for scripts.
